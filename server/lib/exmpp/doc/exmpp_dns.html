Description

SRV lookup support This module provides functions to return an ordered list of servers for a particular SRV service at a domain. This list is randomized according to the weights and priorities in the SRV record.

As a convenience, if SRV records are not found, a one element list of the Domain and Default port will be returned. This allows the function to also return useful data when the domain does not support SRV, for instance at localhost.
