商城商店

点击主界面右下角的商城按钮，即可打开商城界面，此时主界面各功能仅商城及充值按钮可点击，再次点击商城按钮或点击商城界面上的关闭按钮可以退出商城界面。

乐园商城

       商城界面

      

选择拆分：拆分物品可以在背包或者仓库中进行，但若物品原在背包中，拆分时也必须点击背包中的空格，若物品原在仓库中，拆分时必须点击仓库中的空格。

点击背包中的物品拆分按钮，如图：    ，鼠标会变成     状态，此时若左键点击堆叠的物品则可进行拆分；或按住shift键的同时点击堆叠的物品也可直接弹出拆分界面。（仓库中没有专用的拆分按钮，只可点击背包中的拆分按钮或直接按shift点击物品进行拆分。

取消拆分：在弹出拆分界面之前左键点击空白处或右键点击任何地方或再次点击拆分按钮，均可取消拆分状态。

在点击堆叠物品弹出拆分界面之后，点击拆分界面右上角的关闭按钮，即可取消拆分。
